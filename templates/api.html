Your Unique Developer API Key

Everybody using our API is required to use a valid Developer API Key. You automatically get a key when you become a member. Please login to your account, and check your dashboard to find your Developer API Key.


GET request:

With this API, you can obtain a user's all submitted blog in json format. You just need to send a valid GET request to the following URL. 

"https://writeitblog.herokuapp.com/api/blog/"
Include all the following GET parameters when you request the URL:
  1. apikey -this is your API Developer Key
  1. username -Your Username


GET request Example:

Following code is in Python3:

import requests
url="https://writeitblog.herokuapp.com/api/blog/"
params={
    "apikey":"Your_Developer_API_Key",
    "username":"Your Username"
}
r=requests.get(url,params=params)
print(r.content)

POST request:

Creating a new blog via our API is very easy. You simply have to send a valid POST request to the URL shown below. Please make sure you are sending the data as the UTF-8 charset! 

"https://writeitblog.herokuapp.com/api/blog/"
Include all the following POST parameters when you request the URL:
  1. apikey- this is your API Developer Key
  2. username- Your Username
  3. title- Title of Blog
  4. subtitle- Subtitle
  5. author- Author Name
  6. blogtext- Text

Few points to remember while sending a request:

  • Your title must be unique i.e. there should be no other submission of same title with your key or userid.


POST request Example:

Following code is in Python3: 

import requests
url="https://writeitblog.herokuapp.com/api/blog/"
data={
    "apikey":"Your_Developer_API_Key",
    "username":"Your Username",
    "title":"Hello_world",
    "subtitle":"world",
    "author":"james",
    "blogtext":"World is a beautiful place."
    ""
}
r=requests.post(url,data=data)
print(r.content)


PATCH request:

With this API you can update your submitted code. You will need to send a valid PATCH request to the URL below to update the data: 

"https://writeitblog.herokuapp.com/api/blog/"
Include all the following PATCH parameters when you request the URL:
  1. apikey- this is your API Developer Key
  2. username- Your username
  3. otitle- Old Title of existing blog i.e. blog to be updated.
  4. ntitle- New Title
  5. subtitle- Subtitle
  6. author- Author Name
  7. blogtext- Text


PATCH request Example:

Following code is in Python3: 

import requests
url="https://writeitblog.herokuapp.com/api/blog/"
data={
    "apikey":"Your_Developer_API_Key",
    "username":"Your username",
    "otitle":"Hello_world",
    "ntitle":"Hello Universe",
    "subtitle":"Universe",
    "author":"james",
    "blogtext":"Universe is beautiful. "
    ""
}
r=requests.patch(url,data=data)
print(r.content)


DELETE request:

With this API you can delete your submitted code. You will need to send a valid DELETE request to the URL below to delete the data: 

"https://writeitblog.herokuapp.com/api/blog/"
Include all the following DELETE parameters when you request the URL:
  1. apikey- this is your API Developer Key
  2. username- Your Username
  3. title- Title of existing post i.e. post to be deleted.


DELETE request Example:

Following code is in Python3: 

import requests
url="https://writeitblog.herokuapp.com/api/blog"
data={
    "apikey":"Your_Developer_API_Key",
    "username":"Your Username",
    "title":"Hello Universe"
}
r=requests.delete(url,data=data)
print(r.content)
